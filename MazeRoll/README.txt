# 🎮 3D Labirent Top Oyunu (Unity)

Bu proje, bir yazılım mühendisliği öğrencisi olarak Unity kullanarak geliştirdiğim 3D bir labirent oyunudur. Oyunda oyuncu, topu kontrol ederek belirli bir süre içinde finish noktasına ulaşmaya çalışır.

## 🚀 Oyun Özellikleri
- 3D labirent ortamı
- Duvarlara çarpınca can kaybı (3 can hakkı)
- Süre dolmadan finish noktasına ulaşma gerekliliği
- Kazanma ve kaybetme durumları
- Basit kullanıcı arayüzü (UI)

## 📚 Teknolojiler
- Unity Engine
- C#

## 🎓 Hakkımda
Bu oyun, yazılım mühendisliği eğitimim sırasında öğrendiğim Unity becerilerimi geliştirmek amacıyla yapılmıştır. Geri bildirimlere açığım ve yeni şeyler öğrenmeyi seviyorum.

## 🔄 Planlanan Geliştirmeler
- Mobil destek
- Skor sistemi
- Farklı seviyeler

## 🧠 Nasıl Oynanır?
- Topu yön tuşları veya WASD ile hareket ettir.
- Duvarlara çarpmamaya çalış.
- Bitirme noktasına süre dolmadan ulaş.

# 🎮 3D Maze Ball Game (Unity)

This is a 3D maze game developed using Unity as part of my learning journey as a Software Engineering student. The player controls a rolling ball and must reach the finish point before time runs out.

## 🚀 Game Features
- 3D maze environment
- The ball loses a life when it hits walls (3 lives in total)
- Time limit to reach the finish point
- Win/Loss conditions
- Simple UI elements (health, timer)

## 📚 Technologies Used
- Unity Engine
- C# Scripting

## 🎓 About Me
This game was developed by a Software Engineering student to practice and improve Unity development skills. Open to feedback and always learning!

## 🔄 Planned Improvements
- Mobile support
- Score system
- Multiple levels

## 🧠 How to Play
- Move the ball using arrow keys or WASD
- Avoid hitting walls (you lose lives)
- Reach the finish area before time runs out
